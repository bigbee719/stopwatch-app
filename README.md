# ⏱️ Stopwatch App

シンプルで使いやすいストップウォッチアプリケーションです。

## 機能

- **スタート/ストップ**: タイマーの開始と停止
- **リセット**: タイマーを初期状態に戻す
- **ラップ**: ラップタイムを記録
- **リアルタイム表示**: 時間:分:秒の形式で表示

## 使い方

1. 「スタート」ボタンを押してタイマーを開始
2. 「ストップ」ボタンで一時停止
3. 「ラップ」ボタンでラップタイムを記録
4. 「リセット」ボタンで初期状態に戻す

## ローカルで実行

```bash
python3 -m http.server 8000
```

ブラウザで `http://localhost:8000` にアクセスします。
